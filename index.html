📱 How to Set Up

  1. In your Apps Script project, go to Deploy → New deployment
  2. Select type Web app
  3. Set "Execute as" to your email
  4. Set "Who has access" to Only myself
  5. Click Deploy
  6. Copy the Deployment URL from the dialog
  7. Paste it in the Configuration field above
  8. Tap the Install App button above

🛠️ Manual Install Methods

iPhone/iPad (Safari):

  1. Tap the Share button (square with arrow)
  2. Scroll down and tap "Add to Home Screen"
  3. Name it "Flight Control"
  4. Tap "Add"

Android (Chrome):

  1. Tap the menu (3 dots)
  2. Tap "Install app" or "Add to Home Screen"
  3. Confirm

Desktop (Any Browser):

  1. Bookmark this page (Ctrl/Cmd + D)
  2. Or keep it open in a tab for quick access
